# Assessment Policy Guide

## Overview

Assessments are an integral part of the learning experience, designed to evaluate understanding and reinforce learning. This guide explains how assessments work on our platform.

## Types of Assessments

### Knowledge Checks
Quick, ungraded quizzes embedded within lessons to help learners self-assess understanding. These do not count toward final grades.

**Characteristics:**
- 3-5 questions per check
- Immediate feedback provided
- Can be retaken unlimited times
- Not graded

### Module Quizzes
Graded assessments at the end of each module to evaluate comprehension of module content.

**Characteristics:**
- 10-20 questions
- Multiple choice, multiple select, or true/false
- Passing score: 70%
- Limited attempts (typically 3)
- Contributes to final course grade

### Assignments
Practical tasks requiring application of learned concepts. Assignments may include coding exercises, written reports, or project deliverables.

**Characteristics:**
- Detailed rubric provided
- Peer review or instructor grading
- Submission deadline
- Resubmission allowed before deadline

### Capstone Projects
Comprehensive projects at the end of a course or program that demonstrate mastery of skills.

**Characteristics:**
- Real-world problem solving
- Multiple deliverables
- Presentation component (some courses)
- Instructor evaluation

### Proctored Exams
High-stakes exams conducted under supervision (online or in-person) for certification courses.

**Characteristics:**
- Identity verification required
- Time-limited
- No external resources allowed
- Single attempt

## Grading Policy

### Grade Weights
Typical course grade distribution:
- Module Quizzes: 30%
- Assignments: 40%
- Capstone Project: 30%

### Passing Criteria
- Minimum overall score: 60%
- All mandatory assignments submitted
- Capstone project completed (if applicable)

### Grade Scale
- A: 90-100%
- B: 80-89%
- C: 70-79%
- D: 60-69%
- F: Below 60%

## Attempt Policies

### Quiz Attempts
- Most quizzes allow 3 attempts
- Highest score is recorded
- 24-hour waiting period between attempts

### Assignment Resubmission
- Unlimited resubmissions before deadline
- Latest submission is graded
- No resubmission after deadline

## Academic Integrity

All assessments must be completed independently unless explicitly stated as group work. The platform employs:
- Plagiarism detection software
- Browser lockdown for proctored exams
- Random question pools for quizzes

Violations may result in:
- Assessment score of zero
- Course failure
- Account suspension

## Accommodations

Learners requiring accommodations should contact support before starting assessments. Available accommodations include:
- Extended time
- Screen reader compatibility
- Alternative formats
