# Course Structure Guide

## Overview

Our EdTech platform organizes learning content in a hierarchical structure designed for progressive skill development.

## Learning Hierarchy

### Programs
Programs are the highest level of organization. A program typically represents a complete learning path toward a specific career goal or certification. Programs contain multiple courses that build upon each other.

**Example Programs:**
- Full Stack Web Development Program
- Data Science Professional Program
- Cloud Computing Specialist Program

### Courses
Courses are comprehensive learning units focused on a specific subject area. Each course contains multiple modules and typically takes 4-12 weeks to complete depending on the learning pace.

**Course Components:**
- Course overview and objectives
- Prerequisites (if any)
- Module list with estimated duration
- Assessment schedule
- Certification criteria

### Modules
Modules are thematic units within a course. Each module covers a cohesive set of topics and includes various learning activities.

**Module Structure:**
- Learning objectives
- Video lectures
- Reading materials
- Practice exercises
- Module assessment

### Lessons
Lessons are the smallest learning units. Each lesson focuses on a specific concept or skill and typically takes 15-45 minutes to complete.

**Lesson Types:**
- Video lessons (5-20 minutes)
- Interactive tutorials
- Reading assignments
- Hands-on labs
- Knowledge checks

## Course Navigation

### Dashboard
The dashboard provides an overview of your enrolled courses, current progress, and upcoming deadlines. You can access all your learning materials from here.

### Course Page
Each course page displays:
- Course description and instructor information
- Module outline with completion status
- Progress percentage
- Next recommended lesson
- Discussion forum access

### Learning Path
The learning path shows the recommended sequence of courses within a program. While some courses can be taken in parallel, others have prerequisites that must be completed first.

## Prerequisites

Some courses require completion of prerequisite courses or demonstration of prior knowledge. Prerequisites are checked automatically when you attempt to enroll.

**Prerequisite Types:**
- Required course completion
- Skill assessment score
- Prior certification

## Time Commitment

- **Self-paced courses**: Complete at your own pace with suggested timelines
- **Cohort-based courses**: Fixed schedule with specific start and end dates
- **Deadline-flexible courses**: Self-paced with soft deadlines for assessments
